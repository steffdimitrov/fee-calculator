### Fee Calculator

#### Task
https://gist.github.com/mariusbalcytis/1ccd3d4a91b31bb6cc3fd2498edc657e

#### What you need
* UNIX shell.
* PHP >= 7.2
* PHP bcmath extension.

#### PreSetup

```shell
$ git clone git@github.com:steffdimitrov/fee-calculator.git
$ composer install
```

#### How to run the example

```shell
$ php calculate.php input.csv
```

#### Running tests

```shell
$ /vendor/bin/phpunit
```
